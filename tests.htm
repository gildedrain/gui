Tooltips

  • Default This is a default tooltip.
  • Info Use this type of tooltip when you want to provide extra information, like a definition of an academic term.
  • Help Use this type of tooltip when you want to provide helpful tips, formatting advice for form controls, etc.
  • Warning Use this type of tooltip when it's important that the user understand the importance of their actions within the given context.
  • Critical Use this type of tooltip when an error has occurred, or when something is very important within the given context.
  •  Use this when you don't want an underline type tooltip, but a ? icon instead
